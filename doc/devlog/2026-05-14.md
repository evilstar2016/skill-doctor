## Completed Today

- Released the current code as `v0.2.0` and completed the F2 remediation text path so conflict analysis can emit actionable fix suggestions.
- Implemented F5 trust/provenance metadata in `scan`, `show`, and `audit`, surfacing source repository, author, install source, scope, and confidence.
